# sccfm_network_object (Resource)

Provides a Network Object resource. Use this resource to create, read, update, and delete network objects in the Security Cloud Control unified object store.

## Example Usage

```terraform
resource "sccfm_network_object" "example_host" {
  name        = "web-server"
  description = "Web server IP address"
  value       = "10.0.0.1"
}

resource "sccfm_network_object" "example_subnet" {
  name        = "internal-network"
  description = "Internal subnet"
  value       = "10.10.0.0/24"
}

resource "sccfm_network_object" "example_range" {
  name        = "dhcp-range"
  description = "DHCP address pool"
  value       = "192.168.1.100-192.168.1.200"
}

resource "sccfm_network_object" "example_ipv6" {
  name        = "ipv6-host"
  description = "IPv6 host address"
  value       = "2001:db8::1"
}

resource "sccfm_network_object" "with_overrides" {
  name        = "app-server"
  description = "Application server with per-target overrides"
  value       = "10.0.0.50"

  overrides = [
    {
      target_id = "6dae078f-65a8-4cfc-8af8-9554976b5aae"
      value     = "10.1.0.50"
    },
    {
      target_id = "1ba692de-8166-4cee-90b5-6f817c3c2008"
      value     = "10.2.0.50"
    }
  ]
}
```

## Schema

### Required

- `name` (String) The name of the network object.
- `value` (String) The value of the network object. This can be an IP address (e.g., `10.0.0.1`), a CIDR block (e.g., `10.0.0.0/24`), an IPv6 address (e.g., `a:b:c::1`), or a range (e.g., `10.0.0.1-10.0.0.10`).

### Optional

- `description` (String) A human-readable description of the network object.
- `overrides` (List of Object) List of per-target overrides. Each override replaces the default content for its target. Each object has the following attributes:
  - `target_id` (String, Required) The ID of the target (device, service, or shared policy).
  - `value` (String, Required) The override network value for this target.

### Read-Only

- `id` (String) The unique identifier of the network object. This is a UUID generated by the object service.
