# sccfm_service_group (Resource)

Provides a Service Group resource. Use this resource to create, read, update, and delete service groups in the Security Cloud Control unified object store. A service group can contain inline service values and/or references to existing service objects.

## Example Usage

```terraform
resource "sccfm_service_object" "http" {
  name     = "HTTP"
  protocol = "TCP"
  value    = "80"
}

resource "sccfm_service_group" "web_services" {
  name        = "web-services"
  description = "HTTP and HTTPS services"
  referenced_object_uids = [sccfm_service_object.http.id]

  values = [
    {
      protocol = "TCP"
      value    = "8080"
    }
  ]

  overrides = [
    {
      target_id = "6dae078f-65a8-4cfc-8af8-9554976b5aae"
      referenced_object_uids = [sccfm_service_object.http.id]
      values = [
        {
          protocol = "TCP"
          value    = "9090"
        }
      ]
    }
  ]
}
```

## Schema

### Required

- `name` (String) The name of the object.

### Optional

- `description` (String) A human-readable description of the object.
- `overrides` (List of Object) List of per-target overrides. Each override replaces the default content for its target. Each object has the following attributes:
  - `target_id` (String, Required) The ID of the target (device, service, or shared policy).
  - `values` (List of Object, Optional) Inline service values for this target override. Each object has:
    - `protocol` (String, Required) The protocol (e.g., `TCP`, `UDP`, `ICMP`).
    - `value` (String, Optional) The port value (e.g., `80`, `8000-8080`).
  - `referenced_object_uids` (Set of String, Optional) Set of UIDs of service objects for this target override.
- `referenced_object_uids` (Set of String) Set of UIDs of service objects referenced by this group.
- `values` (List of Object) Inline service values included in the group. Each object has:
  - `protocol` (String, Required) The protocol (e.g., `TCP`, `UDP`, `ICMP`).
  - `value` (String, Optional) The port value (e.g., `80`, `8000-8080`). Required for TCP/UDP protocols.

### Read-Only

- `id` (String) The unique identifier of the object. This is a UUID generated by the object service.
