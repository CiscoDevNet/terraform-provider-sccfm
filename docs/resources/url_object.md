# sccfm_url_object (Resource)

Provides a URL Object resource. Use this resource to create, read, update, and delete URL objects in the Security Cloud Control unified object store.

## Example Usage

```terraform
resource "sccfm_url_object" "example" {
  name        = "cisco-website"
  description = "Cisco corporate website"
  url         = "https://www.cisco.com"
}

resource "sccfm_url_object" "with_overrides" {
  name        = "app-portal"
  description = "Application portal with per-target overrides"
  url         = "https://portal.example.com"

  overrides = [
    {
      target_id = "6dae078f-65a8-4cfc-8af8-9554976b5aae"
      url       = "https://portal-staging.example.com"
    }
  ]
}
```

## Schema

### Required

- `name` (String) The name of the object.
- `url` (String) The URL value of the object (e.g., `https://www.example.com`).

### Optional

- `description` (String) A human-readable description of the object.
- `overrides` (List of Object) List of per-target overrides. Each override replaces the default content for its target. Each object has the following attributes:
  - `target_id` (String, Required) The ID of the target (device, service, or shared policy).
  - `url` (String, Required) The override URL value for this target.

### Read-Only

- `id` (String) The unique identifier of the object. This is a UUID generated by the object service.
