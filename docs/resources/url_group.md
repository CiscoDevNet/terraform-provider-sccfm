# sccfm_url_group (Resource)

Provides a URL Group resource. Use this resource to create, read, update, and delete URL groups in the Security Cloud Control unified object store. A URL group can contain inline URL values and/or references to existing URL objects.

## Example Usage

```terraform
resource "sccfm_url_object" "cisco" {
  name = "cisco"
  url  = "https://www.cisco.com"
}

resource "sccfm_url_group" "allowed_sites" {
  name        = "allowed-sites"
  description = "Allowed website URLs"
  referenced_object_uids = [sccfm_url_object.cisco.id]
  values      = ["https://www.example.com"]

  overrides = [
    {
      target_id = "6dae078f-65a8-4cfc-8af8-9554976b5aae"
      values    = ["https://staging.example.com"]
      referenced_object_uids = [sccfm_url_object.cisco.id]
    }
  ]
}
```

## Schema

### Required

- `name` (String) The name of the object.

### Optional

- `description` (String) A human-readable description of the object.
- `overrides` (List of Object) List of per-target overrides. Each override replaces the default content for its target. Each object has the following attributes:
  - `target_id` (String, Required) The ID of the target (device, service, or shared policy).
  - `values` (List of String, Optional) Inline URL values for this target override.
  - `referenced_object_uids` (Set of String, Optional) Set of UIDs of URL objects for this target override.
- `referenced_object_uids` (Set of String) Set of UIDs of URL objects referenced by this group.
- `values` (List of String) Inline URL values included in the group.

### Read-Only

- `id` (String) The unique identifier of the object. This is a UUID generated by the object service.
