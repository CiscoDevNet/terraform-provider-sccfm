# sccfm_network_group (Resource)

Provides a Network Group resource. Use this resource to create, read, update, and delete network groups in the Security Cloud Control unified object store. A network group can contain inline network values and/or references to existing network objects.

## Example Usage

```terraform
resource "sccfm_network_object" "web_server" {
  name  = "web-server"
  value = "10.0.0.1"
}

resource "sccfm_network_group" "servers" {
  name        = "all-servers"
  description = "All server addresses"
  referenced_object_uids = [sccfm_network_object.web_server.id]
  values      = ["10.0.0.100", "10.0.0.101"]

  overrides = [
    {
      target_id = "6dae078f-65a8-4cfc-8af8-9554976b5aae"
      values    = ["10.1.0.100", "10.1.0.101"]
      referenced_object_uids = [sccfm_network_object.web_server.id]
    }
  ]
}
```

### Optional

- `description` (String) A human-readable description of the object.
- `overrides` (List of Object) List of per-target overrides. Each override replaces the default content for its target. Each object has the following attributes:

  - `values` (List of String, Optional) Inline network values for this target override.
  - `referenced_object_uids` (Set of String, Optional) Set of UIDs of network objects for this target override.
- `referenced_object_uids` (Set of String) Set of UIDs of network objects referenced by this group.
- `values` (List of String) Inline network values (IP addresses, CIDR blocks, or ranges) included in the group.
